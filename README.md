# Lost in Cosmos    
My Fourth Published Game     
Free to PLAY!    
You can download this game here or from my itch.io page [osaidii.itch.io](https://osaidii.itch.io)  
This is a small Space Shooter game about fixign your comms to return home!
    
Controls   
Mouse Cursor – Rotate  
LMB - Move   
RMB - Shoot   

Disclosure  
A Little bit of ai has been used in the development of this game.
  

The source files of this game are also present in the same repository!  
These files are meant to be used in Godot Engine!
